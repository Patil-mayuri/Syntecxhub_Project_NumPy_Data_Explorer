# Syntecxhub_Project_NumPy_Data_Explorer

## Project Overview
This project demonstrates NumPy fundamentals using a real-world student performance dataset.

## Features Implemented
- Array creation and slicing
- Mathematical operations
- Axis-wise statistical analysis
- Reshaping and broadcasting
- Save and load NumPy arrays
- Performance comparison with Python lists
- Real CSV dataset handling

## Dataset
Student performance dataset with:
- 15 Students
- 3 Subjects (Math, Science, English)
- Attendance %

## Tools Used
- Python
- NumPy

## How to Run
1. Install NumPy:
   pip install numpy
2. Run:
   python numpy_data_explorer.py
